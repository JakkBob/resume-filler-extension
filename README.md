# 简历表单填充助手

一款智能的微软 Edge 浏览器扩展，支持自动填充和手动提示两种模式，帮助您快速填写简历表单。

## 功能特点

### 双模式填充

1. **自动填充模式**
   - 一键扫描页面所有输入框
   - 智能匹配关键词并自动填充
   - 支持 Vue/React 等前端框架

2. **手动提示模式**
   - 点击输入框时自动显示匹配项
   - 类似浏览器原生地址栏下拉提示
   - 点击选择即可快速填充

### 智能匹配

- 支持多种匹配方式：placeholder、label、name、id、aria-label 等
- 支持多关键词匹配，扩大匹配范围
- 模糊匹配算法，提高识别准确率

### 配置管理

- 可视化配置界面
- 支持导入/导出 JSON 配置
- 预置常用简历字段

## 安装方法

1. 下载或克隆本项目到本地
2. 打开 Edge 浏览器，访问 `edge://extensions/`
3. 开启右上角的"开发人员模式"
4. 点击"加载解压缩的扩展"
5. 选择本项目文件夹

## 使用说明

### 自动填充

1. 点击浏览器工具栏中的插件图标
2. 确保"自动填充模式"已开启
3. 点击"一键自动填充"按钮
4. 插件将自动识别并填充匹配的表单字段

### 手动填充

1. 确保"手动提示模式"已开启
2. 点击网页上的任意输入框
3. 如果有匹配项，将显示下拉菜单
4. 点击菜单项即可填充

### 配置管理

1. 点击插件弹窗中的"配置管理"按钮
2. 在配置页面中可以：
   - 新增/编辑/删除填充项
   - 搜索已有配置
   - 导入/导出 JSON 配置

## 配置格式

```json
[
  {
    "id": "1",
    "keywords": ["姓名", "名字", "您的称呼"],
    "value": "张三"
  },
  {
    "id": "2",
    "keywords": ["手机", "手机号", "电话"],
    "value": "13800138000"
  }
]
```
